## const

-   cannot be changed

## let

-   can be changed later



```javascript
const a = 5;
const b = 2;
let myName = "Hanju";

console.log(a + b);
console.log(a * b);
console.log(a / b);
console.log("Hello! " + myName);

myName = "LiJell";

console.log("your new name is " + myName);
```
